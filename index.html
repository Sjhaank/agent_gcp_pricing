üìñ Parameter Descriptions & Cost Details

üì¶ Website Pages in Data Store

Total number of web pages crawled and indexed in your Vertex AI Data Store. Google allocates exactly 500 KiB (0.000488 GiB) per page for storage calculations, regardless of actual page size.

üí∞ Cost Impact: Storage cost = Pages √ó 500 KiB √∑ 1,048,576 √ó $5/GiB/month
Example: 10,000 pages = 4.88 GiB = $24.41/month
üìè Average Page Size (KB)

The typical size of pages on your website. This parameter is for reference only and does NOT affect pricing. Google uses a fixed 500 KiB allocation per page regardless of actual content size.

‚ö†Ô∏è No Cost Impact: Whether your pages are 50 KB or 2 MB, Google bills the same 500 KiB per page
üîç Monthly Agent Queries

Number of user queries your ADK agent processes per month. Each query triggers: (1) Data Store search, (2) LLM processing to generate response. This is the primary usage driver.

üí∞ Cost Impact: Search cost + LLM cost per query
Example: 25,000 queries = $100 search + $262 LLM = $362/month

üí∞ Cost Impact: 2.67x price difference between editions

The Gemini model your ADK agent uses to generate responses. Set via model="gemini-2.5-flash" in your Agent() constructor.
